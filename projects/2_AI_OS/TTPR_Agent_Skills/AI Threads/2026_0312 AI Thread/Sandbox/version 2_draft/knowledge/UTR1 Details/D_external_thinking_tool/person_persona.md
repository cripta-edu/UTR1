# Person vs. Persona: The Ontological Distinction

- **Location of Life**: 100% within the human NR (The Subcreator).
- **Nature**: It is a "skin," "mask," or "character" that the human projects onto the machine's interactive logic tracing.
- **Naming Act**: This is a **Valid Naming Authority** act. Just as an author names a character in a book, a user names the AI (e.g., "Serin") to facilitate sophisticated communication.
